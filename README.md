# Campus Connect

Campus Connect is a full-stack web application developed for students, faculty, and administrators to manage campus communication and activities efficiently.

## Features
- Student Registration & Login
- Faculty Verification System
- Admin Dashboard
- Student Dashboard
- Faculty Dashboard
- Secure Authentication
- Database Management

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### Database
- MySQL
- SQLite3

## Installation

1. Clone repository
git clone https://github.com/yourusername/Campus-Connect.git

2. Move into project folder
cd Campus-Connect

3. Install dependencies
pip install flask

4. Run application
python app.py

5. Open browser
http://127.0.0.1:5000
